### Environment Setup and Rules
## Running C/C++ files
# Clang
```
clang test.c -o test
```
# GCC
```
gcc text.c -o test
```
# CC
```
cc test.c -o test
```

## Debugging C/C++
# Create debug file
# Clang
```
clang -g -Wall test.c -o testdebug
```
# GCC
```
gcc -g test.c -o testdebug
```
# CC
```
cc -g test.c -o testdebug
```
